# 🎮 Shooting Star Game 

> **Play the Live Demo:** [Playable Browser Build on itch.io](https://chungchehsiao.itch.io/shooting-star)

## 📌 Project Overview

This repository contains the source code for **Shooting Star**, a 3D bullet hell game built in Unity. 

I developed this game for a class in 2017 to practice core programming concepts and game mechanics. I’ve preserved it here to keep a working web demo of a completed project.

## 🖼️ Game Scenes

Lasers only trigger a hit box once the beam thickens.
![Lazer view](lazer-spell.gif)  

Obstacle collisions reduce player lives. Activating the shield ('X' key) prevents damage but consumes 5,000 score points.
![hit view](hit-protect.gif)  

## 🛠️ Tools & Technologies 

*   **Language:** C#
*   **Engine:** Unity 2017
*   **Deployment:** WebGL for browser-based play

## 🕹️ Playable Features

*   **Gameplay Mechanics:** Programmed player movement and obstacle collision detection.
*   **Game UI:** Implemented real-time score tracking, boss HP meters, and stage progression indicators.
*   **Dynamic Difficulty:** Designed an adaptive scaling system that increases the game's difficulty as the boss's health decreases.

## ⚠️ Legacy Code Warning (Archived/Deprecated)

**Please note:** This repository is an archived project built on an older version of Unity. Because of deprecated dependencies, it won't compile correctly in modern Unity editors. Please use the above web demo instead.
